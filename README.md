# Sistema Integral de Gestión de Clientes, Servicios y Reservas

## Descripción

Proyecto desarrollado para la Fase 4 del curso Programación, código 213023, de la Universidad Nacional Abierta y a Distancia.

El sistema permite gestionar clientes, servicios y reservas para la empresa ficticia Software FJ. La solución está desarrollada en Python y aplica los principios de programación orientada a objetos, incluyendo abstracción, encapsulación, herencia, polimorfismo y manejo avanzado de excepciones.

## Servicios disponibles

- Reserva de salas.
- Alquiler de equipos.
- Asesorías especializadas.

## Características

- Registro y validación de clientes.
- Creación de diferentes tipos de servicios.
- Gestión de reservas.
- Confirmación y cancelación de reservas.
- Cálculo de costos, impuestos y descuentos.
- Excepciones personalizadas.
- Registro de eventos y errores en archivos de logs.
- Simulación de operaciones válidas e inválidas.
- Funcionamiento sin base de datos.
